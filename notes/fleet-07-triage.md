## The 2 new signature groups — both folded

- **`_Py_atomic_store_ssize_relaxed | dictiter_iternextitem_lock_held`** (12 veh:
  multiprocessing_connection/context, xml_dom_xmlbuilder, …) → **TSAN-0026** (dict
  lock-free FT iterator). The report is `next()` on a **shared dict iterator**
  (`dictiter_iternextitem_lock_held ← dictiter_iternext_threadsafe ←
  dictiter_iternextkey ← builtin_next`) — the shared-cursor face the op-(h)
  iterator sharing drives. TSAN-0026 already carries the **load** counterpart
  (`_Py_atomic_load_ssize_relaxed | dictiter_iternextitem_lock_held`); this is the
  **store** side of the same word. Folded.

- **`RotatingTree_Add | _PyMem_DebugRawFree`** (1 veh: cProfile) → **TSAN-0008**
  (`_lsprof` not thread-safe, known-area-residual). The report is profiler
  teardown freeing the rotating tree (`_PyMem_DebugRawFree ← freeEntry ←
  RotatingTree_Enum ← clearEntries ← profiler_dealloc:984`) racing a concurrent
  `RotatingTree_Add`. TSAN-0008 already carries the `RotatingTree_Get` counterpart
  and the `profiler_dealloc`/`clearEntries` teardown faces. Colesbury WONTFIX
  (#125165 / devdanzin #126884). Folded.
